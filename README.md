# 🗺️ Maga Dige 

> **"Partner for every journey in Sri Lanka"**

Maga Dige is a modern, full-stack web application designed to help travelers and locals discover essential services, budget-friendly spots, and emergency assistance across Sri Lanka. Whether you're looking for a quick bite, a reliable garage, or a budget stay, Maga Dige provides a seamless and interactive experience.

## ✨ Key Features

* **🌍 Multilingual Support:** Switch effortlessly between Sinhala, English, and Tamil to make the app accessible to everyone.
* **🌗 Premium Dark/Light Mode:** Beautiful, smooth transitions between themes to reduce eye strain, featuring an animated toggle switch.
* **🎨 Modern UI/UX:** Built with stunning "Glassmorphism" design principles, liquid background effects, and highly responsive layouts.
* **📍 Local Services Directory:** Find essential categories quickly:
  * 🍲 Budget Eats
  * 🔧 Garages & Mechanics
  * 🏨 Budget Stays
  * 🛒 Grocery Stores
* **🗺️ Interactive Navigation:** Intuitive map interface to locate exactly what you need, when you need it.

## 🛠️ Tech Stack

### Frontend
* **React.js** (Functional Components, Hooks)
* **Vite** (Next-generation frontend tooling)
* **React Router DOM** (For smooth page transitions without reloading)
* **Context API** (For global state management like Theme and Language)
* **Custom CSS3** (Advanced Glassmorphism & Animations)

### Backend
* Node.js / Express (API Services)
